# Bot of Common Prayer (bcp-bot)
A discord bot that provides resources relating to the Book of Common Prayer!


# Features

## Collect of the Day
Sends the collect(s) of the day at a specific time in a channel.

Configurable:
- Status (enable/disable)
- Message Channel

Future Additions:
- Specified time
- Specified BCP version
- Images for special collects

## More features to come!

# Commands

**`!help`**
- Displays commands and their syntax
- Usage: `!help`

**`!dailycollect`**
- Displays information about daily collects, as well as enable/disable daily collects
- Usage: `!dailycollect <enable/disable/status>`

**`!setchannel`**
- Sets the channel for collect of the day as the current channel
- Usage: `!setchannel`

**`!settime`** 
- Sets the time the collect of the day is sent (Note: only accepts 24-hr EST time)
- Usage: `!settime <time>`

**`!prefix`**
- Sets the prefix for the server. Default is '!'
- Usage: `!prefix <prefix>`
